# Инструкция 1.1: Навигация по файловой системе

Навигация по файловой системе — это базовый навык администратора Linux.  
Знание команд позволяет быстро ориентироваться в каталогах, создавать папки и управлять файлами.

---

## Определение текущей директории

Для определения текущей директории выполните команду:

```bash
pwd
```

Система выведет текущую директорию, например:
```bash
/home/user
```
## Просмотр содержимого папки

Для вывода списка файлов и папок выполните команду:
```bash
ls
```
Подробный вывод с правами, владельцами и датами:
```bash
ls -l
```
Показать скрытые файлы:
```bash
ls -a
```

💡 <u>Совет</u>: используйте <code>ls -lh</code> для отображения размеров файлов в человекочитаемом виде (KB, MB).

## Перемещение между директориями

Перейти в папку:
```bash
cd имя_папки
```
Перейти на уровень выше:
```bash
cd ..
```
Перейти по абсолютному пути:
```bash
cd /путь/к/директории
```
💡 <u>Подсказка</u>: <code>cd ~</code> возвращает в домашнюю директорию пользователя.

## Создание новых папок

Создать одну папку:
```bash
mkdir имя_папки
```
Создать несколько папок сразу:
```bash
mkdir папка1 папка2 папка3
```
Создать вложенные папки за один раз:
```bash
mkdir -p папка1/подпапка1/подпапка2
```
💡 <u>Совет</u>: флаг <code>-p</code> удобен для создания глубоко вложенных директорий за один шаг.
