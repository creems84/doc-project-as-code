*Пример работы с файлами в Linux*

## Создание файла

Для создания файла выполните команду:
```bash
touch имя_файла
```
💡 <u>Совет</u>: <code>touch</code> создаёт пустой файл или обновляет время последнего изменения существующего файла.

## Копирование файла

```bash
cp источник назначение
```
💡 <u>Совет</u>: используйте <code>cp -r</code> для копирования директорий вместе с их содержимым.

## Перемещение или переименование файла

```bash
mv источник назначение
```
💡 <u>Примечание</u>: если путь назначения — существующая директория, файл будет перемещён в неё; если указано новое имя — файл переименуется.

## Удаление файла

```bash
rm имя_файла
```
💡 <u>Осторожно</u>: команда удаляет файлы без возможности восстановления. Для подтверждения используйте <code>rm -i</code>.

## Полезные расширения
Создание нескольких файлов одновременно:

```bash
touch файл1 файл2 файл3
```
Копирование с сохранением атрибутов:

```bash
cp -p источник назначение
```
Принудительное удаление без предупреждения:

```bash
rm -f имя_файла
```
